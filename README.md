# 物流快递仓库管理系统 java680

## 项目概述

本项目是基于 Springboot 和 Vue 的前后端分离的物流快递仓库管理系统。系统结合了现代物流管理理念，利用信息技术手段，实现高效的仓库快递处理流程，提高物流效率。

## 技术栈

- 前端：Vue.js
- 后端：Spring Boot
- 数据库：MySQL

## 功能模块

系统包含以下核心功能模块：

1. 用户管理
   - 用户注册
   - 用户登录
   - 用户权限管理
2. 仓库管理
   - 快递入库
   - 快递出库
   - 库存查询
3. 快递跟踪
   - 快递状态更新
   - 历史轨迹查询
4. 报表统计
   - 快递量统计
   - 库存量统计

## 系统角色

- 系统管理员
- 仓库操作员
- 客户

## 运行环境

- Java 1.8 或更高版本
- MySQL 5.7 或更高版本
- Node.js 12.0 或更高版本

## 部署步骤

1. 准备开发环境：安装上述所需软件。
2. 数据库搭建：根据项目文档建立数据库，执行 SQL 脚本。
3. 后端部署：导入后端代码到开发工具，配置好相应的数据库连接信息，打包部署。
4. 前端部署：在本地安装 Node.js 和 npm，执行 npm install 安装依赖，然后运行 npm run build 打包前端文件，部署至 web 服务器。
